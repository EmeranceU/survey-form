# Bootcamp Survey Form

A responsive survey form built for the SheCanCode bootcamp, collecting participant feedback.

## Features

- Fields for name, email, age, cohort, satisfaction rating, topics learned, and comments
- Form validation with required fields
- Styled radio buttons, checkboxes, and a textarea
- Gradient submit button with hover effect
- Responsive layout for mobile screens (480px and below)

## Built With

- HTML5
- CSS3

## How to Use

Open `index.html` in any browser — no setup required.
